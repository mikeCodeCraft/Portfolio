Project Overview

The Background Remover is a Python application that leverages AI to automatically remove backgrounds from images with a simple drag-and-drop interface. Built with Tkinter and the rembg library, it provides an easy way to process single images or entire folders of images.

This tool was developed to simplify the process of creating transparent PNGs from regular images, eliminating the need for complex photo editing software. It's particularly useful for e-commerce product images, profile pictures, and graphic design projects.

The application features a clean interface with multiple input methods (drag-and-drop, file browser, folder browser) and automatically saves processed images with "_no_bg" appended to the filename.

Installation & Usage

Prerequisites

  • Python 3.6 or higher
  • Tkinter (usually included with Python)
  • For drag-and-drop support: tkinterdnd2 (optional)

Installation

# Clone the repository
git clone https://github.com/yourusername/background-remover.git
cd background-remover

# Install dependencies
pip install rembg pillow tkinterdnd2

How to Use

  1. Run the application: python background_remover.py
  2. Drag and drop an image or folder onto the window
  3. OR use the buttons to select an image or folder
  4. Processed images will be saved automatically with "_no_bg" suffix
  5. The processed image will open in your default image viewer

Input Handling

def handle_input(path):
    """Handle both file and folder inputs"""
    if os.path.isdir(path):
        # Process all images in folder
        files = [f for f in os.listdir(path) 
                if f.lower().endswith(('.png', '.jpg', '.jpeg'))]
        for file in files:
            input_file = os.path.join(path, file)
            output_file = os.path.join(path, f"{os.path.splitext(file)[0]}_no_bg.png")
            process_image(input_file, output_file)
    elif os.path.isfile(path):
        # Process single file
        filename_wo_ext = os.path.splitext(os.path.basename(path))[0]
        output_file = os.path.join(os.path.dirname(path), f"{filename_wo_ext}_no_bg.png")
        process_image(path, output_file)
    else:
        messagebox.showwarning("Invalid Input", "Please provide a valid file or folder.")
