Project Overview

The QR Code Generator is a Python application built with Tkinter that allows users to create customized QR codes with various features including logo embedding, color customization, and support for different QR code types (URL, WiFi login, email, WhatsApp).

This project was developed to provide an easy-to-use tool for generating professional QR codes without requiring online services. It offers local processing, ensuring data privacy and the ability to work offline.

The application includes advanced features like error correction, high-resolution output, and the ability to save generated QR codes in PNG format. It's designed to be intuitive while providing powerful customization options.

Key Features

Multiple QR Types

Supports various QR code types including URLs, WiFi login credentials, email addresses, and WhatsApp numbers.

Logo Embedding

Option to embed a company logo or icon in the center of the QR code for branding purposes.

Color Customization

Customize both foreground and background colors of the QR code for better visual integration.

Error Correction

Built-in error correction (H level) ensures QR codes remain scannable even when partially damaged.

Installation & Usage

Prerequisites

  • Python 3.6 or higher
  • Tkinter (usually included with Python)
  • Additional packages: qrcode, Pillow

Install Dependencies

pip install qrcode[pil] pillow

Running the Application

# Clone the repository
git clone https://github.com/yourusername/qr-code-generator.git
cd qr-code-generator

# Run the application
python qr_generator.py

How to Use

  1. Select the QR code type from the dropdown menu
  2. Enter the required data in the input field
  3. (Optional) Upload a logo and customize colors
  4. Click "Generate QR Code" to create your QR
  5. Use "Save QR Code" to save the generated image

QR Code Generation Function

def generate_qr():
    global generated_qr_img  # Store globally for save button
    data_type = qr_type_cb.get()
    user_data = text_entry.get()
    
    if not user_data:
        messagebox.showwarning("Input Error", "Please enter required data.")
        return

    # Format data based on type
    if data_type == "URL/Text":
        qr_data = user_data
    elif data_type == "WiFi Login":
        try:
            ssid, password = user_data.split(",", 1)
            qr_data = f"WIFI:T:WPA;S:{ssid};P:{password};;"
        except ValueError:
            messagebox.showwarning("Input Error", "Please enter SSID and password separated by a comma.")
            return
    elif data_type == "Email":
        qr_data = f"mailto:{user_data}"
    elif data_type == "whatsapp":
        qr_data = f"https://wa.me/{user_data}"
    else:
        qr_data = user_data

    # Create QR with error correction
    qr = qrcode.QRCode(
        version=None,  # Automatic version sizing
        error_correction=qrcode.constants.ERROR_CORRECT_H,
        box_size=10,
        border=4
    )
    qr.add_data(qr_data)
    qr.make(fit=True)

    # Apply custom colors
    fill_color = fg_color_var.get() or "black"
    back_color = bg_color_var.get() or "white"
    qr_img = qr.make_image(fill_color=fill_color, back_color=back_color).convert('RGB')

    # Embed logo if available
    if logo_path:
        try:
            logo = Image.open(logo_path)
            logo_size = 60
            logo = logo.resize((logo_size, logo_size), Image.LANCZOS)

            qr_width, qr_height = qr_img.size
            pos = ((qr_width - logo_size) // 2, (qr_height - logo_size) // 2)
            qr_img.paste(logo, pos, mask=logo if logo.mode == 'RGBA' else None)
        except Exception as e:
            messagebox.showerror("Logo Error", f"Failed to embed logo: {e}")

    # Display QR in GUI
    fixed_size = 250
    qr_img_resized = qr_img.resize((fixed_size, fixed_size), Image.LANCZOS)
    qr_img_tk = ImageTk.PhotoImage(qr_img_resized)
    qr_label.config(image=qr_img_tk)
    qr_label.image = qr_img_tk

    generated_qr_img = qr_img  # Save original for saving

Logo Upload Function

def upload_logo():
    global logo_path
    path = filedialog.askopenfilename(
        filetypes=[("Image Files", "*.png *.jpg *.jpeg *.bmp")]
    )
    if path:
        logo_path = path
        logo_label.config(text=f"Logo selected ✔️")
